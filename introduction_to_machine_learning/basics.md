##Types of ML Systems

ML systems fall into one or more categories based on how they learn to make predections or generate content

##SuperVised Learning

Supervised learning models can make predections after seeing lots of data with the correct answers ans then discovering the connection between the elements in the data that produce the correct answers , the most two use cases for supervised learning are regression and classification

##regression

> a regression model predects a numeric value ( no labels defined )

##classification

> classification models predict the liklihood that something belongs to a category , its output is a value that states whether or not something belongs to a particular category

classification models are devided into two groups : binary and multiclass classification , binary classification model output a value from a class that contains only two values ( 1 or 0 ) , while multiclass classification model output a value from a class that contains more than two values

##Unsupervised Learning

A unsupervised learning model aims to identify meaningful patterns in a dataset , and many of this models rely on a technique called "clustering" to orgnize data into groups

> Clustering differs from classification because the categories aren't defined by you

##Reinforcement learning

Reinfrorcement learning focuses on how agents can learn to make decisions through trial and error to maximize cumulative rewards . RL allows machines to learn by interacting with an environment and receiving feedback based on their actions. This feedback comes in the form of rewards or penalties.

##Generative AI

generative ai is a class of models that creates content from user input (image , text, audio , video .... ) , and it can takes a variety of inputs to makes a variety of outputs
